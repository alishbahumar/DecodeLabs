# DecodeLabs
Cartify – E-Commerce Mobile App UI/UX Design This repository contains my UI/UX design project completed as part of the CodeAlpha UI/UX Design Internship. The project, Cartify, is a complete e-commerce mobile application design created from concept to interactive prototype using Figma. 📱 About the Project Cartify is designed to give users a smooth, modern, and intuitive online shopping experience. The project covers the full UI/UX design process — from research and user flow planning to wireframing, high-fidelity UI design, and interactive prototyping. 🔍 What's Included

User research and competitor analysis of popular e-commerce apps User flow and information architecture Low-fidelity wireframes High-fidelity UI screens (color palette, typography, icons, components) Interactive clickable prototype in Figma

🛠️ Tools Used

Figma

🎯 Objective To apply core UI/UX design principles — visual hierarchy, consistency, spacing, and typography — while building a realistic, end-to-end mobile app design that demonstrates practical product design skills.
